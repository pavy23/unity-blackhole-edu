# CosmosEdu

**Unity 6 + URP**로 만든 실시간 우주 전시 — 완전 나레이션이 붙은 교육 시뮬레이션 3종
(일반상대성이론 블랙홀 · 우리은하 · 태양계)이 서로 연결되어 있고, 각각 데스크톱 쇼케이스와
Quest 패스스루(MR) 버전을 갖추고 있습니다.

*구 **BlackHoleEdu** — 프로젝트가 첫 이름보다 커졌습니다.*

*[English README](README.md)*

## 전시 구성 한눈에

타이틀 화면(빌드 인덱스 0)에서 언어와 체험을 고르고 시작합니다. 어느 씬에서든 타이틀로
돌아올 수 있습니다 (데스크톱 **F10**, MR은 메뉴 버튼).

| 전시 | 데스크톱 씬 | MR 씬 |
|---|---|---|
| 타이틀 / 선택 | `TitleScreen` | — |
| 블랙홀 | `BlackHoleShowcase` | `BlackHoleMR` |
| 우리은하 | `MilkyWayShowcase` | `MilkyWayMR` |
| 태양계 | `SolarSystemShowcase` | `SolarSystemMR` |

전시들은 이야기로도 이어져 있습니다: 블랙홀 씬은 궁수자리 A*이고, 은하 씬에서 은하 중심으로
다이빙하면 블랙홀 전시로 착지하며(F9), 태양계 투어의 마지막은 은하가 보이는 곳으로 돌아갑니다.

- **4개 언어** — 한국어 · English · 日本語 · 中文: UI·자막·라벨·나레이션(언어별 뉴럴 TTS) 전부. 언제든 전환 (K 또는 화면 선택기)
- **거의 전부 프로시저럴** — 스카이박스·은하·별 표면·사운드스케이프·중력파 처프 오디오까지 코드 생성.
  외부 에셋은 행성/달 실측 텍스처 맵과 번들 CJK 폰트뿐
- **PC(주 타깃) · Quest 패스스루 · WebGL** 지원 (전 셰이더 `target 3.5`; 웹은 URP/WebGL FSR 제한으로 포스트프로세싱 비활성)

## 1 · 블랙홀 (`BlackHoleShowcase` / `BlackHoleMR`)

화면의 모든 것 — 그림자·광자 고리·렌즈 상 — 은 셰이더에서 **매 픽셀 측지선 방정식을
수치적분**한 결과입니다. 밝기와 색은 상대론적 편이(도플러 비밍 · 중력 적색편이 · 흑체복사)로
계산됩니다.

- **슈바르츠실트 레이마칭** — 널 측지선 leapfrog 적분, 얇은 강착원반 + 대기 헤이즈, I ∝ (δ·g)³ 상대론적 비밍, T ∝ r^(−3/4) 박원반 온도, 플랑크 궤적 흑체색
- **커(회전) 블랙홀** — 커–실트 좌표 해밀토니안 적분, D자 그림자, 프레임 드래깅, ISCO(a) 자동 추적 (3키, 스핀 0→0.998)
- **이중 블랙홀 병합** — GW150914 시네마틱 (F4): 원반이 흩어지며 **별빛을 렌징하는 두 개의 맨 블랙홀**이 Peters 궤도로 나선 강하 — 실제 GW150914처럼 가스 없이 — 궤도 동기 처프 오디오, 사중극 변형 중력파 파면, 링다운, 커 잔해(질량 95% · 스핀 0.69)
- **체험 콘텐츠** — 11단계 나레이션 가이드 투어 (F1), 블랙홀 탄생 인트로 (F2), 전 구간 나레이션 1인칭 낙하 체험 (F3, 지평선 안쪽까지 물리적으로 정직한 결말)
- **현상 토글** — 광자 궤적 발사 (Space), 아인슈타인 링 (E), 스파게티화 (T), 상대론적 제트 (J), 중력렌즈 돋보기 (G), 광도 곡선 (V), EHT 관측사진 비교 (4) — 켤 때마다 설명 카드 표시
- **이론 배경 패널** — 상황에 맞는 지배 방정식 카드 (X, 고급 난이도(C)에서 자동 표시)
- **MR 버전** — 방 안의 블랙홀을 잡고 옮기고 크기 조절, 분광형별 별 공 던져 먹이기, 병합 햅틱(맨 블랙홀이 렌징할 별밭을 위해 방이 잠시 우주로 바뀜), 손바닥 미니 블랙홀

| 분류 | 키 |
|---|---|
| 체험 | **F1** 가이드 투어(N/B) · **F2** 탄생 · **F3** 낙하 · **F4** 병합 · **F9** 우리은하 전시로 · **Esc** 건너뛰기/중단 |
| 블랙홀 | **1** 원반 색상 · **2** 질량 · **3** 스핀 · **4** 관측사진 비교 |
| 현상 | **Space** 광자 · **E** 아인슈타인 링(A/D) · **T** 스파게티화 · **J** 제트 · **G** 렌즈 · **V** 광도곡선 |
| 시스템 | 우클릭 회전 · 휠/W/S 줌 · **R** 리셋 · **L** 라벨 · **I** 정보 · **X** 수식 · **U** 몰입 · **M** 소리 · **K** 언어 · **P** 성능 · **F10** 처음으로 · **F12** 스냅샷 · **H** 도움말 · **C** 난이도 |

## 2 · 우리은하 (`MilkyWayShowcase` / `MilkyWayMR`)

하이브리드 은하: **레이마치 볼륨 모델**(막대·밀도파 나선팔·먼지 소광·HII 영역·따뜻한 벌지)에
먼지가 실제로 감쇠시키는 **~48만 개 베이크드 점별**을 겹칩니다. 나레이션 체험 9종:

- **F1 줌 여행** — 태양계에서 은하 전체까지, "우리는 여기" 링과 함께
- **F2 밤하늘** — 밤 언덕에서 이륙해, 밤하늘의 은하수 띠가 원반을 안에서 옆으로 본 모습임을 확인
- **F3 안드로메다 조우** — 2025년 결과(충돌 확률 약 50%의 동전 던지기 — 옛 "확정"이 아님)로 서술.
  M31 곁에서 우리은하를 바라보는 시점, 조석 꼬리, 밀코메다, 그리고 시간 되감기
- **F4 은하 투어** — 7정거장 해부학 수업 (벌지와 막대 · 밀도파 나선팔 · 먼지 레인 · 별 탄생 지대 · 태양의 자리 · 헤일로와 암흑물질)
- **F5 우주 줌아웃** — 국부은하군 → 우주 거대구조 (필라멘트를 따라 5만 2천 개 임포스터 은하)
- **F6 태양계 투어** · **F7 회전 곡선 실험실** (암흑물질의 증거, 인터랙티브) · **F8 은하 동물원** (허블 분류의 볼륨 표본들)
- **F9 궁수자리 A\* 크로스오버** — S별 무리를 지나 은하핵으로 다이빙, 블랙홀 전시로 착지
- **MR 버전** — 지름 ~1.1m 미니어처 은하를 잡고 돌리고 양손으로 크기 조절. 나선면이 보이도록 관람자 쪽으로 기울임, 부위 이름표, 금색 태양 링, 카메라 대신 하이라이트 링이 가리키는 가이드 투어

## 3 · 태양계 (`SolarSystemShowcase` / `SolarSystemMR`)

**실측 텍스처 맵**(행성·달·토성 고리 스트립)과 실제 자전축 기울기·역행 자전, 행성색 혜성꼬리
궤도선, 보정된 시간을 갖춘 정밀 오러리: 자전과 대기 흐름은 정직한 공용 시계(실제 1시간 ≈ 1.5초),
궤도는 가독성 시계(케플러 비율 유지)로 돕니다.

- **행성 클릭**으로 줌인; **F1 행성 투어** — NASA 팩트 스트립이 붙은 9정거장 나레이션
- **F2 진짜 크기** — "보기 좋은 지도"의 고백: 궤도는 AU 비례로, 천체는 실제 비율로 —
  태양계가 사실은 거의 완벽하게 텅 빈 공간임이 드러납니다
- **대기 역학** — 목성의 줄무늬 시어와 대적점 소용돌이(사진 맵 위 유계 2위상 플로우 맵), 금성 초회전, 얼음행성 바람, 지구 구름 표류 — 전부 실제 풍속을 전시 시계로 환산
- **MR 버전** — 방 크기 오러리(해왕성 궤도 ≈ 1.2m)를 잡고 크기 조절; 고정 관람자를 위해 재연출한 진짜 크기 수업: 리그 자체가 축소되어 해왕성 궤도는 제자리에 남고 행성들은 모래알로 사라집니다

## 실제로 반영된 물리 (블랙홀)

화면의 기하는 실제 방정식의 수치해입니다 — 렌즈 효과를 "그려 넣은" 것이 아닙니다:

| 항목 | 수식 | 비고 |
|---|---|---|
| 빛의 휘어짐 | 널 측지선 ODE d²**x**/dλ² = −(3/2)h²**x**/r⁵ (GM=c=1) | 픽셀마다 leapfrog 적분. 그림자(b_crit = 3√3 GM/c² ≈ 2.6 Rs)·광자 고리·아인슈타인 링·원반 위아래 아치는 전부 이 적분에서 **저절로 나타나는** 결과 |
| 커 시공간 | 커–실트 해밀토니안 Φ = H(1+l·q)² | 지평선 r₊ = M+√(M²−a²), 순행 ISCO는 Bardeen–Press–Teukolsky 공식 |
| 비밍·적색편이 | δ = 1/(1−β·cosθ), g = √(1−Rs/r), I ∝ (δg)³, T_obs = T·δg | 원반 좌우 밝기 차는 계산 결과이지 텍스처가 아님 |
| 원반 온도 | Shakura–Sunyaev T ∝ r^(−3/4), 플랑크 궤적 흑체색 | 10 M☉ 기준 ~1200만 K, 질량 프리셋 간 T ∝ M^(−1/4) |
| 시간 지연 | 두 시계 모두 √(1−Rs/r) | 관찰자 시계는 카메라 실거리 사용, "먼 우주 1시간 = X분"은 정적 관찰자 기준 정확한 값 |
| 실제 스케일 | Rs = 2.953 km × M/M☉, 그림자 지름 = 5.2 Rs | 패널의 10 M☉/궁수자리 A*/M87* 수치는 실제 값 |
| 쌍성 나선 강하 | Peters 감쇠 a(t) = a_f+(a₀−a_f)(1−t/T)^¼, ω² = M/a³, f_GW = 2f_궤도 | 잔해 질량 95%·스핀 0.69는 GW150914 측정값 |
| 광자 발사기 | 동일한 측지선 ODE를 CPU에서 적분 | 임계거리 안쪽 포획은 연출이 아닌 계산 결과 |

다른 전시의 수치도 정직합니다: 은하의 비례(원반 반지름 ~16 kpc, 태양 8.2 kpc, 막대 ~27°),
회전 곡선 실험실의 평탄한 곡선 vs 케플러 예측, 태양계 팩트 스트립·자전축·역행 자전,
2025년 안드로메다 확률 — 전부 실제 값입니다.

## 간소화·연출된 것들

엄밀하지 **않은** 부분을 정직하게 밝힙니다:

- **시간·공간은 어디서나 압축.** 병합은 수개월을 ~40초로(케플러 비율 유지), 원반·행성 궤도는 가독성 시계로 과속, 안드로메다 조우는 ~100억 년을 압축. 질량 프리셋의 수치는 정확하지만 화면상 크기 비율은 연출.
- **쌍성 렌징은 두 슈바르츠실트 굴절의 중첩.** 두 블랙홀의 해석적 계량은 존재하지 않고 마지막 궤도들은 수치상대론의 영역. 링다운 오디오는 감쇠 사인파(준정규 모드 스펙트럼 아님). 중력파 "고리"는 보이지 않는 변형률의 시각화.
- **복사 전달 없음.** 원반 밝기, 은하의 방출/소광 모델, 난류 노이즈와 밝은 매듭은 물리가 빚은 프로시저럴 아트이지 MHD/RT 시뮬레이션 출력이 아님. 블룸·노출은 가독성 위주 튜닝.
- **낙하 등 시네마틱은 연출**이며 고유시간 적분이 아님 (자막의 *서술*은 올바른 물리).
- **스파게티화**는 가독성용 캡 스트레치 (실제 조석 기울기는 Δa ∝ 1/r³); 초신성·제트·인트로는 물리적 동기가 있는 VFX.
- **은하 조우는 안무된 모프**이지 N체 시뮬레이션이 아님; 조석 꼬리·위상 혼합은 실제 타임라인을 따르는 셰이더 변형.
- **행성 대기 움직임**은 실제 풍속으로 사진을 흘리는 플로우 맵 — 패턴 속도는 실제, 유체역학은 시뮬레이션 아님.
- 커 모드는 순행 적도 박원반만 표시; 제트는 장식 (Blandford–Znajek 없음).
- MR 별 공 궤도는 방 스케일 GM의 뉴턴 역학.

앱 내 이론 카드(X)마다 해당 주제가 계산인지 연출인지 명시되어 있습니다.

## 수업 활용 시 참고

- **스톱워치 말고 숫자를 믿으세요.** 패널 값(Rs·그림자 크기·온도·지연 배율·행성 데이터)은 정확하고, 화면상 시간·각크기는 압축입니다.
- **시간 지연 시계는 *정지(호버링)* 관찰자 기준.** 궤도·낙하 관찰자는 속도 항이 추가로 필요합니다 — 탐사선이 "블랙홀 곁에 떠 있다"고 서술되는 이유.
- **색은 물리적 색이지만 실제 관측 이미지와 다릅니다:** EHT 사진은 전파 간섭계의 false color이고, 광학으로 보면 눈이 멀 만큼 밝습니다. 비교 모드(4)로 직접 토론해 보세요.
- **지평선 안쪽은 근거 있는 삽화.** 안에서 밖으로는 아무것도 보고할 수 없고, 자막도 그렇게 말합니다 — 그 주장 자체가 물리입니다.
- **안드로메다 이야기는 불확실성 교육에 좋습니다**: 2025년 재분석이 "확정 충돌"을 동전 던지기로 바꿨습니다 — 과학이 스스로를 갱신하는 좋은 예.
- 나레이션 대본 = 화면 자막(4개 언어 모두) — 인용해도 안전하며, 수정 시 edge-tts로 오디오를 재생성하세요.
