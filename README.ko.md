# BlackHoleEdu

실시간 일반상대성이론 기반 블랙홀 교육 시뮬레이션 (Unity 6, URP).

*[English README](README.md)*

화면의 기하(그림자·광자 고리·렌즈 상)는 셰이더에서 **매 픽셀 측지선 방정식을 수치적분**한 결과이며,
밝기와 색은 상대론적 편이(도플러 비밍 · 중력 적색편이 · 흑체복사)로 계산됩니다.

## 주요 기능

- **슈바르츠실트 레이마칭** — 널 측지선 leapfrog 적분, 얇은 강착원반 + 대기 헤이즈, I ∝ (δ·g)³ 상대론적 비밍, T ∝ r^(−3/4) 박원반 온도
- **커(회전) 블랙홀** — 커–실트 좌표 해밀토니안 적분, D자 그림자, 프레임 드래깅, ISCO(a) 자동 추적 (3키, 스핀 0→0.998)
- **이중 블랙홀 병합** — GW150914 시네마틱 (F4): Peters 궤도 나선 강하, 궤도 동기 처프 오디오, 사중극 변형 중력파 파면, 링다운, 커 잔해(질량 95% · 스핀 0.69)
- **체험 콘텐츠** — 11단계 나레이션 가이드 투어 (F1), 블랙홀 탄생 인트로 (F2), 전 구간 나레이션 1인칭 낙하 체험 (F3, 지평선 안쪽까지 물리적으로 정직한 결말)
- **현상 토글** — 광자 궤적 발사 (Space), 아인슈타인 링 (E), 스파게티화 (T), 상대론적 제트 (J), 중력렌즈 돋보기 (G), 광도 곡선 (V), EHT 관측사진 비교 (4) — 켤 때마다 무엇을 보는지 설명 카드 표시
- **이론 배경 패널** — 화면 상황에 맞는 지배 방정식 카드 (X, 고급 난이도(C)에서 자동 표시)
- **4개 언어** — 한국어 · English · 日本語 · 中文: UI·자막·라벨·나레이션(언어별 뉴럴 TTS) 전부. 우측 상단 선택기 또는 K키
- **프로시저럴 에셋** — 별밭/은하수 스카이박스, 별 표면(대류·코로나), 앰비언트 사운드, 처프 오디오 전부 코드 생성 (외부 에셋 0)
- **MR 씬** — Quest 패스스루용 BlackHoleMR: 잡고 옮기고 크기 조절, 별 던져 먹이기, 병합 햅틱, 손바닥 미니 블랙홀

## 실제로 반영된 물리 수식

화면의 기하는 실제 방정식의 수치해입니다 — 렌즈 효과를 "그려 넣은" 것이 아닙니다:

| 항목 | 수식 | 비고 |
|---|---|---|
| 빛의 휘어짐 | 널 측지선 ODE d²**x**/dλ² = −(3/2)h²**x**/r⁵ (GM=c=1) | 픽셀마다 leapfrog 적분. 그림자(b_crit = 3√3 GM/c² ≈ 2.6 Rs)·광자 고리·아인슈타인 링·원반 위아래 아치는 전부 이 적분에서 **저절로 나타나는** 결과 |
| 커 시공간 | 커–실트 해밀토니안 Φ = H(1+l·q)² | 지평선 r₊ = M+√(M²−a²), 순행 ISCO는 Bardeen–Press–Teukolsky 공식 |
| 비밍·적색편이 | δ = 1/(1−β·cosθ), g = √(1−Rs/r), I ∝ (δg)³, T_obs = T·δg | 원반 좌우 밝기 차는 계산 결과이지 텍스처가 아님 |
| 원반 온도 | Shakura–Sunyaev T ∝ r^(−3/4), 플랑크 궤적 흑체색 | 10 M☉ 기준 ~1200만 K, 질량 프리셋 간 T ∝ M^(−1/4) |
| 시간 지연 | 두 시계 모두 √(1−Rs/r) | 관찰자 시계는 카메라 실거리 사용, "먼 우주 1시간 = X분"은 정적 관찰자 기준 정확한 값 |
| 실제 스케일 | Rs = 2.953 km × M/M☉, 그림자 지름 = 5.2 Rs | 패널의 10 M☉/궁수자리 A*/M87* 수치는 실제 값 |
| 쌍성 나선 강하 | Peters 감쇠 a(t) = a_f+(a₀−a_f)(1−t/T)^¼, ω² = M/a³, f_GW = 2f_궤도 | 잔해 질량 95%·스핀 0.69는 GW150914 측정값 |
| 광자 발사기 | 동일한 측지선 ODE를 CPU에서 적분 | 임계거리 안쪽 포획은 연출이 아닌 계산 결과 |

## 간소화·연출된 것들

엄밀하지 **않은** 부분을 정직하게 밝힙니다:

- **시간·공간 압축.** 병합은 수개월의 나선 강하를 ~40초로 압축(케플러 비율은 유지), 원반 회전은 과속(실제 초대질량 블랙홀 원반은 궤도당 수십 분~수 시간이라 정지 화면처럼 보임), 질량 프리셋의 수치는 정확하지만 화면상 크기 비율은 연출.
- **쌍성 렌징은 슈바르츠실트 편향의 중첩 근사.** 두 블랙홀의 해석적 계량은 존재하지 않으며 마지막 궤도는 본래 수치상대론의 영역. 링다운 오디오는 감쇠 사인파(준정규 모드 스펙트럼 아님), 중력파 "링"은 보이지 않는 변형률의 시각화.
- **복사 전달 없음.** 원반 밝기·불투명도 프로파일, 난류 무늬, 밝은 덩어리는 물리(케플러 전단)로 구동되는 절차적 아트이지 MHD 시뮬레이션이 아님. 블룸·노출은 가독성용.
- **낙하 체험은 시네마틱** — 고유시간 적분이 아닙니다. 페이싱·카메라 틸트·지평선 이후의 축소되는 하늘 원은 연출이지만, 자막의 **서술**(빛원뿔 기울어짐, 마지막 빛이 머리 위로, 뒤쪽 하늘은 계속 보임)은 올바른 물리입니다.
- **스파게티화**는 가독성용 (r₀/r)^1.4 스트레치(실제 조석 기울기는 Δa ∝ 1/r³). 초신성·제트·인트로는 물리적 동기를 가진 VFX.
- **커 모드**는 순행 적도 박원반만 표현. 제트는 장식이며 스핀과 연결되지 않음(Blandford–Znajek 미구현), 원반 휨(Bardeen–Petterson) 없음.
- MR 별공 궤도는 방 크기에 맞춘 뉴턴 역학.

인앱 이론 카드(X)에 각 주제가 계산인지 연출인지 명시되어 있습니다.

## 교육 활용 시 주의할 점

- **숫자는 믿고, 스톱워치는 믿지 마세요.** 패널의 수치(Rs, 그림자 크기, 온도, 시간지연 배율)는 정확하지만 화면상 소요 시간·겉보기 크기는 압축된 것입니다. 실제 궁수자리 A*의 그림자는 ~50 마이크로각초 — 어떤 카메라도 26 Rs에서 맴돌 수 없습니다.
- **시간지연 시계는 '정지(호버링) 관찰자' 가정입니다.** 공전·낙하 중인 관찰자는 속도 항이 추가로 필요합니다(그래서 탐사선을 "블랙홀 곁에 떠 있는"으로 서술). 블랙홀에 다가가면 **내 시계도 함께 느려지므로**, 패널은 혼동을 막기 위해 '먼 우주 1시간' 기준으로 전환됩니다.
- **색은 물리적 색조지만 실제 관측 영상과 다릅니다.** EHT 사진은 전파 간섭계의 가짜색이고, 광학으로 보면 눈이 멀 만큼 밝습니다. 비교 모드(4)로 이 차이를 직접 다뤄보세요.
- **지평선 안쪽 장면은 '근거 있는 삽화'입니다.** 안쪽에서는 어떤 정보도 돌아올 수 없고, 자막도 그렇게 말합니다 — 그 주장 자체가 물리입니다.
- 나레이션 대본 = 화면 자막(4개 언어) 그대로이므로 수업 자료로 인용해도 안전합니다. 대본 수정 시 edge-tts로 음성을 재생성하세요.
